First time users

If this is the first time you've run XOWA, then you will need to do some further setup.

  • Import Simple Wikipedia
Simple Wikipedia is a small wiki that fits in 1 GB. It can be setup in about 3 minutes, depending on internet connectivity.
When you are ready to start, click this link:
Set up Simple Wikipedia
Feel free to look around during the import process. You can start at Help:Contents. When the process completes, it will open Simple Wikipedia.
If you encounter problems, please visit Help:Importing the first wiki for more information.
  • Set up images
Image support requires additional programs (ImageMagick and Inkscape). These programs will use another 500 MB and can be setup in about 5 minutes.
When you are ready to start, click one of these links:
Set up images (Windows) If the install fails, please visit Set up images (Windows)
Set up images (Linux)
Set up images (Mac OS X)
  • Set up non-English languages
Non-English language wikis will require extra files. These files will use 45 MB and can be setup in about 1 minute.
When you are ready to start, click this link:
Set up non-English languages If the install fails, please visit Help:Set up languages

Tips and tricks

  • The "<" and ">" symbols in the upper left-hand corner represent the back and forward button. Click on them just like you would in a web browser. You can also use Alt+Left and Alt+Right.
  • Press Ctrl+F on your keyboard. This will take you to the Find box (in the lower left-hand corner). Type "tips". Notice that the word "Tips" is highlighted above. See Help:Find box for more info.
  • Press Alt+D on your keyboard. This will take you to the Address bar (at the top of the screen). Type "s.w:Earth" (no quotes) and press Enter. This will open the Simple Wikipedia page for Earth. See Help:Address bar/Shortcuts for more shortcuts.
  • Copy and paste urls directly into the address bar with the middle mouse button to open the page. For example:
    • Highlight the following url: http://simple.wikipedia.org/wiki/World history
    • Press Ctrl+C
    • Click the middle mouse button in the url bar above. The "World history" page in "Simple Wikipedia" will open.
Now you can copy and paste any Wikipedia url directly into XOWA and its page will open (and all its images will download.) See Help:Address bar/URLs for more info
